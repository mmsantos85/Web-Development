Intro

1. Grootste verschil tussen World Wide Web (WWW) en het internet is dat WWW verwijst naar software zoals bijv. een grote verzameling van webpagina's en het internet verwijst naar hardware. Denk aan een enorme computernetwerkwerk dat de apparaten verbind met bijv. glasvezelkabels.

2. HTML zorgt voor een algemene basis/layout. CSS zorgt voor details. Denk aan elementen, kleuren, lettertypes etc. JavaScript zorgt voor een meer dynamische en interactieve website aan de client zijde.

3. HTML Tags zijn elementen die nodig zijn voor het opmaken van een HTML pagina. Hiervoor gebruik je een openings-tag '<'en een sluit-tag '/>'.Voorbeelden HTML tags:

'p' van Paragraaf is een stuk tekst/alinea.

'header' is voor een header/titel van je document.

'Div' is om delen van een pagina in te sluiten zoals een menu, paragraaf, of een titel.

4. Het 'HTML' element markeert het begin en einde van een HTML document. Hiermee geef je aan dat het document een HTML code bevat. Hierin worden ook de elementen 'HEAD' en 'BODY' in opgenomen. Het 'HEAD' element markeert het begin en einde van dat dat deel van een HTML document waar meestal bijv. de titel staat. Het 'BODY element markeert het begin en einde van een een deel van een HTML document. Door deze drie elementen in de basis op te nemen krijgt het HTML document een overzichtelijke structuur.
