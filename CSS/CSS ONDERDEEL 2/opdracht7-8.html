3. Wat merk je op met width en height?

De width en height is nu meer px dan wat ik heb aangegeven in mijn css box. De padding, margine en border voegen extra px toe aan de box. Dit kan je oplossen met de property Box-sizing.

Opdracht 8. Welke CSS property kun je gebruiken om ervoor te zorgen dat de waarde van de width en height de totale breedte en hoogte is van het blok van de voorgaande opdracht?

Box-sizing, deze property definieert hoe de width en height word gecalculeerd. Hiermee kan je meerdere boxen/eigenschappen samenvoegen en passend maken. Bijv. soms voegen padding, border en margine meer px toe dan wat je hebt aangegeven met de width en height. Box-sizing merged/uitlijnen het als het ware samen tot 1 passend box.
