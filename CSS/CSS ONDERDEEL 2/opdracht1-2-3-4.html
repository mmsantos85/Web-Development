Onderdeel 2 Het box-model

Opdracht 1. Wat zijn de eigenschappen van block boxes en inline boxes?

In CSS hebben we in het algemeen 2 type boxen, block en inline. Deze twee eigenschappen referen naar hoe de box zich navigeert als je kijkt naar page flow, en hoe het correspondeert met andere boxen op een webpage.

Block Box: 1 van de eigenschappen van de block box is dat het alle ruimte in neemt in de container waar het geplaatst word. Hierdoor verschuiven andere elementen/boxes die onder een block box zijn geplaatst. Dit kan je veranderen/aanpassen door property's toe te voegen in je block box in CSS.

Inline Box: In contrast met de block box zijn de eigenschappen van inline box dat het enkel ruimte inneemt die het nodig heeft. Inline boxes verschuift geen andere elementen/boxes die er onder geplaatst zijn.

Opdracht 4. Hieronder zie je een box-model. Leg uit wat de volgende termen betekenen: content, padding, margin en border.

  • Content - De content, inhoud van de box waar o.a. tekst en afbeeldingen komen te staan.
  • Padding - Lege/transparante ruimte rondom de tekst/content binnen een CSS box.
  • Border - Rand dat om de padding en content komt.
  • Margin - Margin is de transparante afstand van de rand van een box ten opzichte van andere HTML elementen.
