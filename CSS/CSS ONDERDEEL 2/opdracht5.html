Opdracht 5. Als je ruimte wilt maken tussen de border en content, welke css property gebruik je dan? Maak een voorbeeld in je antwoord.

Padding - Deze property maakt lege/transparante ruimtes rondom de tekst/content binnen een CSS blok/box. Je kan ook specifiek 1 deel aangeven wat je wilt aanpassen met de eigenschappen padding-top, padding-right, padding-bottom, padding-left
