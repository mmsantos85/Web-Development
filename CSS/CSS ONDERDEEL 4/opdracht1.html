Onderdeel 4 Pseudo Elements

Opdracht 1. Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

In CSS3 kun je gebruik maken van zogenoemde pseudo elementen. Met deze pseudo elementen kun je gedeeltes van HTML elementen van specifieke styling voorzien. Zo zou je bijvoorbeeld van een stuk tekst de eerste regel een andere styling kunnen geven dan de rest van de tekst.

  • Pseudo elementen '::before' en '::after'- Dit zijn twee pseudo elementen die veel gebruikt worden. Je kunt hiermee iets voor en iets na een bepaald element laten weergeven.   • Pseudo element '::selection'- Met deze pseudo element kan je bepaalde geselecteerde delen van je tekst stylen. Stel iemand selecteert een stukje tekst op je site, dan kun je met het pseudo element "selection" bepalen hoe de styling van dit geselecteerde stukje tekst moet zijn.
  • Pseudo element ::first-letter- Met deze element kan je styling toevoegen aan de eerste letter van je geselecteerde tekst die je wilt bewerken.
