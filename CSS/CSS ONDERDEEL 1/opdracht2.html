CSS Syntax

 

 Een CSS Rule-Set bestaat uit een selector en een declaratieblok:

       


De selector wijst naar het HTML-element dat u wilt opmaken.
Het declaratieblok bevat een of meer declaraties gescheiden door puntkomma's.
Elke declaratie bevat een CSS-eigenschapsnaam en een waarde, 
gescheiden door een dubbele punt.Meerdere CSS-declaraties worden gescheiden door puntkomma's,
En declaratieblokken zijn omgeven door accolades.
