Opdracht 3. Er zijn verschillende selectors, voorbeeld:

  • Type selectors
  • Descendant selectors
  • Class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

De type selector selecteerd/target elementen in een HTML document die ik wil aanpassen/stylen. Bijv ik wil alle 'a' tags of 'p' tags van mijn HTML layout in het rood dan kan ik dit in CSS aangeven. Voordeel hieraan is dat je specifiek target.

De CSS Descendant selector matched alle elementen die nakomelingen zijn van een specifieke element. De decendant selector is een onderdeel van de CSS combinators:'A combinator is something that explains the relationship between the selectors'. Selectors die een onderdeel zijn van combinators: descendant selector (space), child selector (>), adjacent sibling selector (+) en general sibling selector (~). Voordeel hiervan is dat je selectors kan combineren om samen een element te targeten.

De Class selector selecteert HTML elementen met een specifieke class attribuut. Bijv. je kan een div een class naam geven met een specifieke naam of verwijzing van je Div. Hier mee kan je in je CSS file specifiek je class vorm geven. In je CSS begin je een class naam altijd met een punt. Het grote voordeel van classes is dat je ze kan hergebruiken op verschillende plaatsen en zelfs verschillende pagina's. Het wordt dan eenvoudig op CSS stijlregels toe te passen op deze klassen en ze een mooie opmaak te geven.
