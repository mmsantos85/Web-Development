Onderdeel 6. Transitions

Opdracht 1. Zoek uit wat transitions zijn en beschrijf met eigen woorden wat het betekent.

Met CSS transitions kan je aanpassingen/veranderingen/effecten toevoegen aan property values zodat ze een geleidelijke verandering/transitie ondergaan doormiddel van een gegeven duratie met tijd.

  • Transition- Een 'shorthand' property waarmee je de vier transition property kan samenvoegen in 1 property.
  • Transition-delay: Hiermee geef kan je vertraging van secondes toepassen aan property's.
  • Transition-duration: Hiermee kan je specifiek aangeven hoeveel secondes of milisecondes het duurt voor transitie compleet is.
  • Transition-property: Geeft specifiek aan de naam van een CSS property waar de toegapste effect voor is.
  • Transition-timing-function- Hiermee kan je specifiek aangeven hoe snel de snelheid moet gaan. Bijv. constante snelheid van begin to eind of langzaam opbouwend etc.
