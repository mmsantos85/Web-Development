Onderdeel 3. Display & Positioning

Opdracht1. Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

  • Static - Deze postion property is gepositioneerd volgens de normale flow van een webpage. De top, right, left en bottom property's hebben geen invloed en dit is dan ook de default value. Statische positionering is de standaard voor elk element. Het betekent gewoon: "zet het element op zijn normale positie in de documentstroom, hier is niets speciaals te zien".
  • Relative - Deze position werkt hetzelfde als de static position. Alleen hier kan je wel top, bottom, left en right property's toevoegen. Hiermee kan je elementen in een box verplaatsen zonder dat het effect heeft op de andere gebruikte elementen.
  • Fixed - de fixed position zorgt ervoor dat een element gefixeerd blijft op 1 plek, ook al scroll je van boven naar beneden en vice versa. De top, bottom, left en right kan je gebruiken om het element te positioneren. Een gefixeerde element laat geen gat achter op de webpage waar het normaliter staat.
  • Absolute - Elementen die relative gepositioneerd zijn gaan mee met de normale flow van de webpage. Voor het tegenovergstelde/contrast effect kan je absolute postion gebruiken. Hiermee word de element uit de normale flow gehaald en positioneerd zich aan het dichtbijzijnde nacestor position. Als een absolute postion geen geposiotioneerde ancestor heeft gebruikt het de body van de webpage en volgt hte normale flow van de webpage.
  • Sticky - Deze position werkt bijna hetzelfde als de absolute position. Verschil is dat het elementen fixeerd gebaseerd op waar de gebruiker scroll positie is.
